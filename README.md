## ✨ Features

### 🚀 Productivity-First
- **🌍 Multi-City Tracking:** Add any number of locations to monitor global time at a glance.
- **🕹️ Time Travel Slider:** Scrub forward or backward in time (±48h) to visualize future or past offsets across all tracked zones.
- **🔍 Smart Search:** Lightning-fast city search powered by Apple's CoreLocation and MapKit.
- **📋 One-Click Summary:** Copy a clean text-based conversion summary directly to your clipboard.

### 🎨 Premium Experience
- **🎨 Custom Branding:** Set your own accent color to match your workspace (or use the system accent).
- **🌗 Context-Aware:** High-quality visual indicators for "Morning", "Afternoon", and "Night" phases.
- **✨ Built with SwiftUI:** A native, lightweight experience designed specifically for macOS Sonoma and later.

### 🛠️ System Integrated
- **🔔 Auto-Updates:** Stay up to date with built-in version checking and one-click upgrades.
- **📍 Smart Home Detection:** Optionally auto-detect your current "Home" location as you travel.

## 📥 Installation

> [!IMPORTANT]
> **Notarization Note:** This app is open-source but not notarized by Apple. On the first launch, **Right-Click → Open** to bypass macOS Gatekeeper.

### Option 1: One-liner Terminal (Recommended)
```bash
curl -fsSLO https://raw.githubusercontent.com/EliKO109/Easy-Time-Zones/main/install.sh && bash install.sh
```

### Option 2: Manual Install
1. Download the latest `EasyTimeZones.dmg` from [Releases](https://github.com/EliKO109/Easy-Time-Zones/releases/latest).
2. Drag **Easy Time Zones.app** to your `/Applications` folder.
3. Right-click and choose **Open** for the first run.

---

## 🔄 Updates & Maintenance

The app checks for updates automatically on launch. When a new version is available, a banner appears in the menu—just click it to download and update.

To update manually, simply re-run the installation script above.

---

## 🛠 Build from Source

```bash
git clone https://github.com/EliKO109/Easy-Time-Zones.git
cd Easy-Time-Zones
open "Easy Time Zones.xcodeproj"
```

Then press **Cmd+R** in Xcode to run.
